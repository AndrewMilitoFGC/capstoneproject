# Text summarization capstone

Flask web app: upload a `.txt` file, get an AI summary via the [Groq](https://console.groq.com/) API.

## Quick start

1. Create a virtual environment and install dependencies (see `Instructions.md`).
2. Copy `.env.example` to `.env` and set `GROQ_API_KEY`.
3. From the project folder, run:

   ```bash
   python app.py
   ```

   Your browser should open to the app. If it does not, visit `http://127.0.0.1:5000/`.

For more detail, see `Instructions.md`.
